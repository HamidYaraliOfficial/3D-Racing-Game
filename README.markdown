# 🇺🇸 English

## Overview

**3D Racing Game** is a modern racing game developed with **Python** and **PyQt6**. The project features a pseudo‑3D road rendering system, dynamic vehicle movement, obstacle avoidance, coin collection, nitro boosts, particle effects, multiple themes, and multilingual support.

The game is designed to demonstrate advanced GUI programming techniques, real‑time animation, custom painting, and game logic implementation using PyQt6.

## Features

### Gameplay
- Smooth pseudo‑3D racing experience
- Progressive difficulty system
- Dynamic speed and acceleration physics
- Nitro boost system
- Multiple lives system
- Coin collection mechanics
- Lap progression system
- Score tracking
- High score recording

### Visual Effects
- Custom 3D road rendering
- Animated scenery and environment
- Day and night themed visuals
- Particle explosion effects
- Nitro visual effects
- Screen shake effects
- Collision flashes
- Animated collectible coins

### User Interface
- Modern HUD display

### Customization

## Game Mechanics

Players control a racing car on a continuously moving road. The objective is to survive as long as possible while:

- Avoiding obstacles
- Collecting coins
- Completing laps
- Increasing score
- Managing nitro usage
- Reaching higher levels

Difficulty gradually increases as the player progresses through laps.

## Project Structure

The project is organized into:

- Translation system
- Theme system
- Game canvas and rendering engine
- Physics and gameplay logic
- HUD and interface components
- Settings management
- Particle and animation systems

## Highlights

- Pure PyQt6 implementation
- No external game engine required
- Clean object‑oriented architecture
- Multilingual support
- Modern visual design

## 项目简介

**3D赛车游戏** 是一个基于 **Python** 和 **PyQt6** 开发的现代赛车项目。游戏包含伪3D道路渲染、动态车辆控制、金币收集、氮气加速、粒子特效、多主题界面以及多语言支持。

该项目展示了如何使用 PyQt6 实现实时动画、游戏逻辑和高级图形绘制技术。

- 合理使用氮气
